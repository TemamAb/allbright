# allbright Documentation

Architecture specs, API docs, deployment guides, and operational runbooks.

## Primary Interface

**Allbright Desktop App**: The official and exclusive interface for all user operations, monitoring, and management. Terminal-based commands are deprecated and disabled.

- Download and install the MSI from `build-desktop-app.bat`
- Launch `Allbright-Desktop.exe`
- Access all features through the desktop dashboard (`allbright-dashboard.html`)
